# Resort Website

A website designed to showcase the services, features, and amenities of a resort. It aims to provide an intuitive experience for users looking for resort information, booking options, and services.

##Live Demo

Check out the live version of the website here: [Live Demo](https://suniltechs.github.io/Resort_website/)


## Features
- Beautiful and responsive UI design.
- Booking system integration.
- Resort information (Rooms, Activities, Dining, etc.).
- Contact and inquiry forms.
- Integration with Google Maps for resort location.

## Technologies Used
- HTML
- CSS
- JavaScript
- Bootstrap (or any other CSS framework)
- Backend (optional) - PHP, Node.js, etc. (If applicable)

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/suniltechs/Resort_website.git
   ```
2. Navigate to the project folder:
   ```bash
   cd Resort_website
   ```
3. Open the `index.html` file in your browser to view the website.
